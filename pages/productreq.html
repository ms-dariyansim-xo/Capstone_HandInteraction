TEAM GOALS AND BUSINESS OBJECTIVES

Break barriers by joining together as one community in creating and collaborating on an application--

An web appliction to start, build, and elevate. We listen and value one's opinion, as well as those that will utilize the product.

Through our vision and setting a positive influence, this could bring awareness to the Deaf/HoH community and the use of American Sign Language through.

BACKGROUND AND STRATEGIC FIT

Accessibility is a huge thing and the need of it is everywhere, especially in the world of technology. This is thought of frequently for future developments that would include avatars, incorporating options for immersed experiences such as basic conversations.

USER STORIES

# Story Title (by users) Description Priority Notes
1 Mobile Application A user wants application on mobile devices. Must Have Prefers to use it "on-the-go" and rarely uses desktop computer.
2 More to learn A user wants more sign to learn. Must Have User would like more than just an "Alphabet" category. Interested in numbers, colors, animals categories.
3 Written language options A user suggested written translated in other languages. Should Have A user is suggesting Spanish, in addition to English. Also hoping to see Sign Language in other countries.

ACCEPTANCE CRITERIA:

Web application is in the works for mobile-preferred users.
